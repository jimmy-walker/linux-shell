# Command
| Bash Shortcuts |                                          |
| -------------- | ---------------------------------------- |
| Ctrl-a         | Move to the start of the line.           |
| Ctrl-e         | Move to the end of the line.             |
| Ctrl-w         | Delete from the cursor to the start of the word. |
